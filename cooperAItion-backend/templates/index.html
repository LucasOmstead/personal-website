About CooperAItion Backend

This Flask API serves as the backend for the CooperAItion prisoner's dilemma tournament. It provides AI model training using simulated annealing algorithms to create competitive strategies.

Key Features:

  • Simulated annealing optimization for strategy training
  • Multiple prisoner's dilemma strategies (Tit-for-Tat, Random, etc.)
  • Customizable payoff matrices
  • RESTful API endpoints for frontend integration

This backend is part of a larger Angular frontend application that provides an interactive gaming experience for exploring game theory concepts.

Quick Start

GET /get_model

Get a trained AI model for the prisoner's dilemma game. This endpoint returns strategy parameters optimized through simulated annealing.

POST /getmodel

Alternative endpoint for retrieving AI models with custom parameters. Send a POST request with your desired configuration.
