Shape Shifters

3D Maze Game with Shape-shifting Mechanics

About the Game

Shape Shifters is an exciting 3D maze game that combines strategy, stealth, and quick reflexes. In this asymmetric multiplayer experience, one player takes on the role of a hunter while the other becomes the hunted - but with a twist.

Game Features

  • Shape-shifting Mechanics: Player 1 can morph into NPCs to blend into the environment and avoid detection
  • 3D Maze Environment: Navigate through complex 3D maze structures with multiple levels and hiding spots
  • Coin Collection: The hunted player must collect coins scattered throughout the maze while avoiding capture
  • Strategic Gameplay: Balance between collecting coins and staying hidden from the hunter
  • Asymmetric Multiplayer: Two distinct player roles with different objectives and abilities

Technical Details

Technology: Java3D, Java

Course: COMP-2800 Software Development

Project Type: Group Project

Key Challenges: 3D rendering, collision detection, AI behavior for NPCs, multiplayer synchronization

Gameplay Overview

The game creates a tense cat-and-mouse dynamic where the hunted player must use wit and timing to outmaneuver their pursuer. The shape-shifting ability adds a layer of strategy, as players must choose the right moments to transform and blend in with the environment.

This project was developed as part of the COMP-2800 Software Development course, where we learned to work as a team to design, implement, and test a complex interactive application. The project challenged us to integrate 3D graphics, game physics, and multiplayer networking into a cohesive gaming experience.
